## 1. Project Overview
This project focuses on building an interactive Power BI dashboard to analyze key performance metrics in the hospitality domain. The dashboard enables stakeholders to monitor revenue trends, evaluate property performance, and identify opportunities to improve operational efficiency.

---

## 2. Business Problem
Hospitality businesses generate large volumes of data across bookings, pricing, and occupancy. However, without proper visualization, it becomes difficult to:
- Track performance across cities and properties
- Identify revenue trends over time
- Compare weekday vs weekend performance
- Evaluate booking platform effectiveness

This dashboard solves these problems by providing a centralized and interactive view of business KPIs.

## 3. Key Metrics
- Revenue: 1.69 Billion
- RevPAR (Revenue per Available Room): 7,337
- ADR (Average Daily Rate): 12.70K
- Occupancy Rate: 57.8%
- DSRN (Daily Sellable Room Nights): 2528
- Realisation Percentage: 70.1%

---

## 4. Dashboard Features

### 4.1 Filters
- City-level filtering
- Room class filtering
- Time-based filtering (Week-wise analysis)

### 4.2 Visualizations
- KPI cards for high-level metrics
- Revenue distribution by category (Luxury vs Business)
- Trend analysis for RevPAR, ADR, and Occupancy
- Property-level performance comparison
- Platform-wise analysis of ADR and Realisation
- Weekday vs Weekend performance breakdown

---

## 5. Key Insights
- Luxury category contributes the majority of revenue (~61.6%)
- Weekend performance is stronger than weekday performance in terms of occupancy and RevPAR
- Cities such as Bangalore and Mumbai generate higher revenue compared to others
- Booking platforms show noticeable differences in pricing (ADR) and realization rates
